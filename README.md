# Zephyr/1.0

Description of this stuff here for people finding us on GitHub or wanting to contribute.



# How to run

Preferred, requires docker:

```
  cd ./server && runlocal.sh
```

Direct:

```
  cd ./server && pip install -r requirements.txt && python3 server.py
```
